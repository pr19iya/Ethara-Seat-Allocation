# Ethara Seat Allocation & Project Mapping System

## Overview

A full-stack seat allocation and project mapping application designed to manage employees, projects, seats, allocations, new joiners, occupancy analytics, and natural-language seat queries.

- Employee management
- Department and project management
- Seat inventory management
- Seat allocation and release
- Prevention of duplicate seat allocation
- Search and filters
- Dashboard analytics
- Floor-wise utilization
- Project-wise utilization
- New joiner allocation
- Rule-based AI assistant
- Responsive frontend
